# Face-Mask-Detection-App
Real-time Face Mask Detection using a Convolutional Neural Network (CNN) with an interactive Streamlit web application for predicting whether a person is wearing a face mask.
Face Mask Detection using CNN and Streamlit

📌 Overview

This project is a Deep Learning-based Face Mask Detection System developed using TensorFlow/Keras and deployed with Streamlit. The application classifies facial images into two categories:

* ✅ With Mask
* ❌ Without Mask

The model is trained on a labeled face mask dataset using a Convolutional Neural Network (CNN) and provides real-time predictions through an easy-to-use web interface.

📂 Dataset

The model is trained using a publicly available Face Mask Dataset containing images of people with and without face masks.

Dataset classes:

* With Mask
* Without Mask
📸 Application Workflow

1. Launch the Streamlit app.
2. Upload a facial image.
3. The image is preprocessed automatically.
4. The CNN model predicts the class.
5. The application displays whether the person is wearing a face mask.
